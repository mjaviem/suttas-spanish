"Bhikkhus, existen estas tres cuestiones.
¿Cuáles tres?
Interés, odio y autoengaño.
Bhikkhus, ciertamente existen estas tres cuestiones.
Bhikkhus, ciertamente para abandonar estas tres cuestiones, estas tres cuestiones deben ser cultivadas.
¿Cuáles tres?
Para abandonar el interés, lo asqueroso debe ser cultivado. Para abandonar el odio, la benevolencia amistosa debe ser cultivada. Para abandonar el autoengaño, el entendimiento debe ser cultivado.
Bhikkhus, ciertamente para abandonar estas tres cuestiones, estas tres cuestiones deben ser cultivadas.
