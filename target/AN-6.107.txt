"Bhikkhus, existen estas tres bases.
¿Cuáles tres?
Interés, odio y ofuscamiento.
Bhikkhus, pues existen estas tres bases.
Pues, bhikkhus, para abandonar estas tres bases deben suscitar estas tres bases.
¿Cuáles tres?
Para abandonar el interés deben suscitar lo asqueroso. Para abandonar el odio deben suscitar la benevolencia amistosa. Para abandonar el ofuscamiento deben suscitar el saber.
Pues, bhikkhus, para abandonar estas tres bases deben suscitar estas tres bases.
