"Bhikkhus, existen estas tres cuestiones.
¿Cuáles tres?
Interés, odio y autoengaño.
Bhikkhus, pues existen estas tres cuestiones.
Pues, bhikkhus, para abandonar estas tres cuestiones deben suscitarse estas tres cuestiones.
¿Cuáles tres?
Para abandonar el interés debe suscitarse lo asqueroso. Para abandonar el odio debe suscitarse la benevolencia amistosa. Para abandonar el autoengaño debe suscitarse el entendimiento.
Pues, bhikkhus, para abandonar estas tres cuestiones deben suscitarse estas tres cuestiones.
