"Bhikkhus, existen estas tres cuestiones.
¿Cuáles tres?
Interés, odio y autoengaño.
Bhikkhus, pues existen estas tres cuestiones.
Pues, bhikkhus, para abandonar estas tres cuestiones deben suscitar estas tres cuestiones.
¿Cuáles tres?
Para abandonar el interés deben suscitar lo asqueroso. Para abandonar el odio deben suscitar la benevolencia amistosa. Para abandonar el autoengaño deben suscitar el entendimiento.
Pues, bhikkhus, para abandonar estas tres cuestiones deben suscitar estas tres cuestiones.
