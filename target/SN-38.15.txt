«Amigo Sāriputta, se llama "cuerpo del ser, cuerpo del ser". ¿Pues qué es, amigo, el cuerpo del ser?»

«Amigo, estos cinco montones de asidero son llamados "cuerpo del ser" por el Dichoso, los siguientes: el montón de asidero de las imágenes, el montón de asidero de las sensaciones, el montón de asidero de las significaciones, el montón de asidero de las constituciones, el montón de asidero de las distinciones. Amigo, pues estos cinco montones de asidero son llamados cuerpo del ser por el Dichoso.»

«Mas, amigo, ¿hay un camino, hay una dedicación, para la comprensión plena de este cuerpo del ser?»

«Hay, pues, amigo, un camino, hay una dedicación, para la comprensión plena de este cuerpo del ser.»

«Mas, amigo, ¿qué camino, qué dedicación, para la comprensión plena de este cuerpo del ser?» «Pues precisamente, amigo, el Camino Óctuple Noble es para la comprensión plena de este cuerpo del ser, lo siguiente: la manera de ver cabal... la compostura cabal. Pues este es, amigo, el camino, esta es la dedicación, para la comprensión plena de este cuerpo del ser.» «Amigo, valioso es el camino, valiosa es la dedicación, para la comprensión plena de este cuerpo del ser. Y es suficiente, mas amigo Sāriputta, para la atención.»
